What ranshous.com wants to be:

  • private
  • easy to use
  • photo and video repository

What ranshous.com does not want to be:

  • annoying
  • flickr
  • intrusive
  • clunky

Why ranshous.com exists:

  • "I'd like to be able to go to ranshous.com and have a log on with password. Once on the site I'd like to be able to access all photos and video anyone put up. It would be good to have some sort of an album system and to see who posted what in case you want to ask them about something." - gary

current feature set:

  • You can upload multiple media (under the same name / album / etc) (images only)
  • You can currently upload media of all common image formats.
  • When uploading new media you can create a new album for that media or add that media to an existing album that you created.
  • All media can be commented on.

planned features:

  • Better views / sorting when browsing sections
  • The ability to edit the properties of your media and user (image / album / etc)
  • The ability to upload multiple media at a time with unique names
  • Styling and interface improvements
  • Get a larger hard drive =]
